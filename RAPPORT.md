J'ai créé un composant fonctionnel `Owl` en React avec JSX.
Ce composant reçoit deux props : `image` et `description`.
Les props permettent de réutiliser le même composant avec des contenus différents.
J'ai placé la logique avant le `return` pour que le code soit plus lisible.
J'ai utilisé le rendu conditionnel : si l'image n'existe pas, un message alternatif s'affiche.
Dans `App.js`, j'affiche plusieurs hiboux avec des descriptions différentes.
Cela démontre la réutilisabilité du composant sans copier le code HTML.
Le code reste simple et adapté à un niveau débutant.
